# Zakat Coin White Paper

This repository contains the official white paper for Zakat Coin (ZKTC).

Zakat Coin is a utility-based blockchain protocol focused on automated,
transparent charitable giving using smart contracts.

This repository serves as the canonical technical and economic reference
for exchanges, data providers, compliance partners, and ecosystem reviewers.

Token: ZKTC  
Network: BNB Smart Chain (BEP-20)  
Website: https://zktcoin.com
